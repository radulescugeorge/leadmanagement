About This Project

This is a demo application as a learning project. It'a simple functional REST API as a sales opportunities management system. Technologies used: Java 21, Spring Boot 3.4.3, Maven, PostgreSQL,Thymeleaf, HTML, Bootstrap.

Features include:

  • ✔ Lead generation and total sales calculation
  • ✔ Product listing and pricing
  • ✔ Customer tracking with loyalty program
  • ✔ Sales agent management
